### Issue 1: Could not locate the Extensions panel

The assignment instructions referred to installing Claude Code and Codex through the Extensions marketplace.

### Investigation

The current version of Cursor uses a newer interface and does not expose the traditional Extensions panel in the same way as older versions.

### Resolution

I searched within Cursor and verified that Claude and Codex are already available as built-in AI models.

### Issue 2: Git command not recognized

Git was not installed on the system.

### Resolution

Downloaded and installed Git for Windows, configured username and email, and verified installation.
